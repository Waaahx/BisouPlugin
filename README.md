# BisouPlugin

## Setup
- **Minecraft versions** : `1.12.*`
- **Required Plugins** : `Skript, skDragons`

Then, you have to put the bisou.sk file into the folder : 
```txt
server/plugins/Skript/script/
```

## Usage

```txt
/bisou <player>
```

## Permissions

| Utility     | Name          | 
| :--------   | :-------      | 
| Use /bisou  | `bisou.bisou` | 
| No cooldown | `bisou.pass`  |
